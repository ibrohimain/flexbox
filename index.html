Flexbox

  • Flexbox (Flexible Box) — bu CSS'ning zamonaviy joylashtirish texnologiyasi bo'lib, veb-sahifadagi elementlarni oson va moslashuvchan tarzda tartibga solishga imkon beradi. Flexbox asosiy maqsadi elementlarni gorizontal yoki vertikal yo'nalishda joylashtirishni soddalashtirish va turli o'lchamdagi ekranlarga moslashadigan tartib yaratishdir. Flexbox'da ikkita asosiy komponent mavjud: konteyner (ota element) va flex elementlar (ichidagi bolalar elementlari).
  • Flexbox afzalliklari:
    •Elementlarni gorizontal yoki vertikal joylashtirishni soddalashtiradi.
    •Har xil ekran o'lchamlari uchun moslashuvchan tartib yaratish imkonini beradi.
    •Elementlar orasidagi bo'sh joylarni avtomatik boshqaradi.
    •Float va position kabi eski texnikalarga ehtiyojni kamaytiradi.
  • Flexboxning asosiy tushunchalari: Flex konteyner: Flexbox'ni ishlatmoqchi bo'lgan elementingizni konteyner deb belgilashingiz kerak. Buning uchun display: flex; xususiyati ishlatiladi. Flex konteyner ichidagi barcha elementlar avtomatik ravishda flex elementlarga aylanadi va tartiblanadi.
  • Asosiy o'q va kesish o'qi: Asosiy o'q (main axis): Flex elementlar asosiy o'q bo'ylab joylashadi.
    Asosiy o'qni gorizontal yoki vertikal qilish uchun flex-direction xususiyatidan foydalaniladi:
    row: Elementlar chapdan o'ngga (gorizontal) joylashadi (standart qiymat).
    column: Elementlar yuqoridan pastga (vertikal) joylashadi.
    Kesish o'qi (cross axis): Asosiy o'qga nisbatan perpendikulyar o'q. Masalan, agar asosiy o'q gorizontal bo'lsa, kesish o'qi vertikal bo'ladi.
  • Joylashuvni boshqarish:

    justify-content: Flex elementlarni asosiy o'q bo'ylab joylashtirish uchun ishlatiladi.
    flex-start: Chapga (yoki yuqoriga) qarab joylashadi.
    flex-end: O'ngga (yoki pastga) qarab joylashadi.
    center: O'rtada joylashadi.
    space-between: Elementlar orasida bo'sh joy hosil qiladi.
    space-around: Har bir elementning atrofida bo'sh joy hosil qiladi.
  • Align-items: Flex elementlarni kesish o'qi bo'ylab joylashtirish uchun ishlatiladi.
    flex-start: Yuqoridan joylashadi (yoki chapdan).
    flex-end: Pastdan joylashadi (yoki o'ngdan).
    center: O'rtada joylashadi.
    stretch: Elementlar konteyner balandligini to'ldiradi.
  • Flex-grow, flex-shrink, va flex-basis: Flex elementlarning o'sishini va qisqarishini boshqaradi.
    flex-grow: Elementning o'sish imkoniyatini belgilaydi.
    flex-shrink: Element qisqaradigan miqyosni belgilaydi.
    flex-basis: Elementning boshlang'ich o'lchamini belgilaydi.
  • Xulosa: Flexbox bilan tartib yaratish veb-sahifani tezroq va moslashuvchan qilishga yordam beradi. Ekranlar o'lchami yoki elementlarning o'zgarishiga mos keladigan dizaynlarni sodda va samarali tarzda yaratishga imkon beradi.

Flexbox parent&child

  • Flexbox tizimida ota element (parent) va bola elementlar (child) o'rtasidagi bog'lanish juda muhim. Flexbox'ning asosiy vazifasi ota element (flex container) ichidagi bola elementlarni (flex items) moslashuvchan va tartibli ravishda joylashtirishdir
  • Keling, ota (parent) va bola (child) elementlarning qanday ishlashini aniqroq tushuntiramiz.
  • Flexbox Ota Element (Parent) Ota element — bu flex container deb nomlanadi. Flexbox faqatgina bu konteynerning ichida joylashgan elementlarga ta’sir qiladi. Ota elementni flex container qilib belgilash uchun CSS'da display: flex; qoʻllanadi. 
  •                             div class="cr"
                        div class="item" 1 div
                        div class="item" 2 div
                        div class="item" 3 div
                        div
                        
                                   .cr{
                                display: flex; /* Bu joyda ota element flex konteyner bo'ladi */
                                }
  • Flexbox Bola Elementlar (Child) Ota element ichidagi har bir bola element flex elementga aylanadi. Flexbox yordamida ularni tartibga solish va moslashuvchan qilib joylashtirish mumkin. Bola elementlar asosiy va kesish o'qlari bo'ylab joylashtiriladi, va ular mos ravishda turli xususiyatlar orqali boshqariladi.
  • Ota Element (Parent) Xususiyatlari flex-direction: Flex elementlarning asosiy o'q yo'nalishini belgilaydi.
    row: Elementlar gorizontal joylashadi (chapdan o'ngga).
    column: Elementlar vertikal joylashadi (yuqoridan pastga).
    row-reverse: Gorizontal joylashish, lekin teskari yo'nalishda (o'ngdan chapga).
    column-reverse: Vertikal joylashish, lekin teskari yo'nalishda (pastdan yuqoriga). 
  •                             .container {
                                display: flex;
                                flex-direction: row; /* yoki column, row-reverse */
                              }
  • justify-content: Flex elementlarni asosiy o'q bo'ylab joylashtiradi.
    flex-start: Elementlar boshidan joylashadi.
    flex-end: Oxirida joylashadi.
    center: O'rtada joylashadi.
    space-between: Elementlar orasida bo'sh joy qoldiradi.
    space-around: Elementlar atrofida teng bo'sh joy hosil qiladi. 
  •                             .container {
                                display: flex;
                                justify-content: center; /* yoki space-between, flex-start va h.k. */
                              }
  • align-items: Flex elementlarni kesish o'qi bo'ylab joylashtiradi (balandlik bo'yicha).
    flex-start: Yuqorida joylashadi.
    flex-end: Pastda joylashadi.
    center: O'rtada joylashadi.
    stretch: Barcha elementlar balandlik bo'yicha cho'ziladi. 
  •                             .container {
                                display: flex;
                                align-items: stretch; /* yoki center, flex-end */
                              }
  • flex-wrap: Bola elementlar konteynerdan chiqib ketmaslik uchun o'ralib joylashishini boshqaradi.
    nowrap: Barcha elementlar bitta qatorga joylashadi (standart qiymat).
    wrap: Elementlar joy yetmay qolsa, yangi qatorga o'tadi.
    wrap-reverse: O'ralish, lekin teskari yo'nalishda. 
  •                             .container {
                                display: flex;
                                flex-wrap: wrap; /* yoki wrap-reverse */
                              }
  • flex-shrink: Bola element kichrayadigan darajani belgilaydi. 1 qiymati element konteyner kichiklashganda ham qisqarishini bildiradi. 
  •                             .item {
                                flex-shrink: 1; /* Element konteyner kichrayganda qisqaradi */
                              }
  • flex-basis: Elementning boshlang'ich o'lchamini belgilaydi (px, %, va h.k.). Bu xususiyat elementning flex konteynerda qanday joylashishini belgilashda boshlang'ich o'lchovni nazorat qiladi. 
  •                             .item {
                                flex-basis: 200px; /* Har bir elementning asosiy kengligi 200px bo'ladi */
                              }
  • align-self: Faqat bitta bola elementning kesish o'qi bo'ylab joylashishini boshqaradi. Bu align-items ga o'xshash, lekin faqat bitta bola elementga ta'sir qiladi. flex-start, flex-end, center, stretch 
  • .item {
                            align-self: center; /* Faqat shu element o'rtaga joylashadi */
                          }
  • div class="cr" div class="item" 1 div div class="item" 2 div div class="item" 3 div div 
        .cr {
        display: flex;
        justify-content: space-around;
        align-items: center;
        flex-wrap: wrap;
      }
      
      .item {
        flex-grow: 1;
        flex-basis: 150px;
        align-self: flex-start;
      }
  • Bu misolda: Ota element (container) flex konteyner bo'lib, bola elementlar o'rtasida teng bo'sh joy yaratadi (space-around). Bola elementlar (item) o'sish imkoniyatiga ega (flex-grow: 1) va har bir element boshlang'ichda 150px kenglik oladi (flex-basis: 150px).
  • Xulosa: Ota element (parent) display: flex; orqali flex konteynerga aylanadi va uning ichidagi har bir element bola element (child) bo'lib, ular flex xususiyatlariga muvofiq joylashadi. Ota element asosiy tartibni va joylashuvni belgilaydi, bola elementlar esa o'lchamlari, joylashuvi, va boshqa xususiyatlari bo'yicha nazorat qilinadi.
